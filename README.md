# Portfolio Page

My personal portfolio site — built to showcase skills, recommendations, contact information and projects in the future.

**See it Live:** [sigge1511.se](https://www.sigge1511.se)

---

## Tech Stack

| Category | Technology |
|---|---|
| Framework | React 19 |
| Language | TypeScript |
| Build tool | Vite |
| Routing | React Router DOM v6 |
| Contact form | EmailJS |
| Sanitization | DOMPurify |
| Testing | Vitest + Testing Library |
| Linting | ESLint |
| Hosting | Strato |
